# Promptdown Contracts API

API REST para gerenciamento de contratos da plataforma Promptdown.

## Visão Geral

Este repositório contém a especificação OpenAPI e documentação para a API de Contratos do Promptdown. Fornece endpoints para criar, ler, atualizar e deletar contratos com suporte a versionamento, autenticação JWT e tratamento robusto de erros.

## Características

- **CRUD Completo**: Create, Read, Update, Delete de contratos

## Estrutura do Projeto

```
.
├── openapi.yaml           # Especificação OpenAPI 3.0
├── README.md             # Este arquivo
├── CHANGELOG.md          # Histórico de versões
├── examples/             # Exemplos de requisições/respostas
└── errors/              # Documentação de códigos de erro
```

## Contribuindo

1. Edite `openapi.yaml` para mudanças na especificação
2. Adicione exemplos em `examples/`
3. Documente erros em `errors/`
4. Atualize `CHANGELOG.md`
